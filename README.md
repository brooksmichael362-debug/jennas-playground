# Jenna's Playground

A browser-based music cutting and mixing app for building dance mixes.

Live site: https://YOUR-USERNAME.github.io/jennas-playground/

## What it does

Import songs, select the sections you want, arrange them on a timeline, add fades and crossfades, and export the finished mix as MP3 or WAV. Everything runs 100% in the browser — audio files never leave your computer.

## Features

- Drag-and-drop import (MP3, WAV, M4A, OGG, AAC, FLAC)
- Visual waveform selection with draggable edges for fine-tuning
- Timeline with stacked clip rows, drag-to-reorder, edge-resize
- Per-clip volume, tempo (50–150%), fade in / fade out
- Crossfade presets (Quick / Smooth / Long)
- Auto-scrolling timeline that follows the playhead during playback
- Export to MP3 (128 / 192 / 320 kbps) or WAV (lossless)
- Works on phone, tablet, and desktop
- Keyboard shortcuts: Space = play/pause, Delete = remove selected clip

## Usage

Open the site, click the `?` icon in the top-right for step-by-step instructions.
